## Steps necessary to run the software

See instructions. **Delete this line and place instructions to download, configure, and run the software here.**

### Database setup

Start up a MongoDB database either using [MongoDB Atlas](https://www.mongodb.com/products/platform/atlas-database) or a locally hosted [Docker container](https://hub.docker.com/_/mongo).

Create a `.env` file that corresponds to the given example file `env.example`. Edit the values to match your database.

### Virtual environment

Create a new virtual environment:
```
python -m venv .venv
```
To activate the virtual environment:  
(Mac/Linux)
```
source .venv/bin/activate
```
(Windows)
```
.venv\Scripts\activate.bat
```
To install the required dependencies into the currently-active virtual environment, run:
```
pip install -r requirements.txt
```
